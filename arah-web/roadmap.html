Notable MVP Stories

The foundation — these stories unlock everything else.

MVP
WEB-001
Admin Login
Firebase Auth sign-in page for admin portal with email/password and Google SSO, redirecting to dashboard on success.
MVP
WEB-002
Dashboard Overview
Top-level KPI cards: active incidents, pending reports, registered users, and platform health — all sourced from Firebase Firestore.
MVP
WEB-011
Report Queue
Paginated list of community reports pending moderation with photo preview, approve/reject buttons, and reason input.
MVP
WEB-023
User List
Searchable and filterable user table with status (active/banned), role, report count, and ban/unban action.
MVP
WEB-035
Flood Zone Map
MapLibre GL admin interface for drawing, editing, and activating flood zones — with severity level selector and broadcast trigger.

Current Focus

All 52 stories are at 🔲 Todo status — we are actively seeking first contributors to pick up MVP stories and start building the admin dashboard foundation.

Start with docs/bmad/04-stories.md in the repository, pick a WEB-00x story, branch off in the format feature/WEB-NNN-description, and open a pull request.
