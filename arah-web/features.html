Tailwind + shadcn/ui — No Custom CSS

All UI in arah-web is built with Tailwind CSS v3 utility classes and shadcn/ui components. No custom CSS files are permitted — contributors must use Tailwind utilities and shadcn primitives exclusively. Dark mode variants (dark:) must be applied alongside all colour utilities.
