Two Portals, One Platform

The web layer that powers moderation, analytics, and the public face of Arah.

🖥️

Admin Dashboard

Report moderation queue, bulk approve/reject, flood zone drawing tool, MAU/DAU analytics, user management with PDPA export, and moderator audit log.

🌐

Public Portal

Landing page with embedded map, real-time API status page, incident feed, and app download links. Optimised for SEO with Next.js App Router.

🔐

Security

Firebase Auth with MFA for admins, role-based access control, IP allowlist, full audit trail, and PDPA-compliant data handling throughout.

Quick Start

Get the dev server running in 2 steps.

# 1. Clone and install
git clone https://github.com/deanz93/arah-web.git
cd arah-web
npm ci

# 2. Start the dev server
npm run dev
# → http://localhost:3000

Need Firebase credentials, environment variables, or emulator setup? See Contributing for the full guide.
