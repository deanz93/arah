3D Maps Guide

Terrain elevation + 3D building extrusion in MapLibre GL — DEM pipeline, fill-extrusion layer config, and camera presets.

Two Layers of 3D

🏔️

3D Terrain

Raster DEM tiles that tilt the ground plane when the map is pitched. Shows hills, valleys, and mountain ranges.

Source: Mapbox Terrain DEM v1
Tile size: 512px
Max zoom: 14
Exaggeration: 1.4×
🏢

3D Buildings

Extruded building volumes from OSM footprints, using building:levels and height tags.

Source: OSM buildings layer
Min zoom: 14
Height: height tag → levels×3.5m → 10m
Style: fill-extrusion

MapLibre Setup

Enable 3D Terrain

map.on('load', () => {
  map.addSource('terrain', {
    type: 'raster-dem',
    url: 'https://tiles.arah.my/terrain-rgb/{z}/{x}/{y}.png',
    tileSize: 512,
    maxzoom: 14,
  });

  map.setTerrain({ source: 'terrain', exaggeration: 1.4 });

  map.addLayer({
    id: 'sky',
    type: 'sky',
    paint: {
      'sky-type': 'atmosphere',
      'sky-atmosphere-sun': [0.0, 90.0],
      'sky-atmosphere-sun-intensity': 15,
    },
  });
});

3D Buildings Layer

map.addLayer({
  id: '3d-buildings',
  type: 'fill-extrusion',
  source: 'arah-tiles',
  'source-layer': 'buildings',
  minzoom: 14,
  paint: {
    'fill-extrusion-color': '#C8C0B8',
    'fill-extrusion-height': [
      'coalesce',
      ['get', 'height'],
      ['*', ['get', 'building:levels'], 3.5],
      10,           // default 10m if no tags
    ],
    'fill-extrusion-base': ['get', 'min_height'],
    'fill-extrusion-opacity': 0.85,
  },
});

Camera Preset for 3D View

map.easeTo({
  pitch: 55,
  bearing: -17.6,
  zoom: 15.5,
  duration: 800,
});

These values work well for KL city centre. Adjust bearing to align buildings toward the viewer.

Terrain DEM Pipeline

scripts/build-terrain.sh
# 1. Downloads Copernicus DEM tiles for Malaysia bbox
#    bbox: 99.6°E 0.8°N → 119.3°E 7.4°N
# 2. Merges into single GeoTIFF (gdal_merge)
# 3. Converts to Terrain-RGB PNG tiles (rio-cogeo + gdal2tiles)
# 4. Packages as PMTiles
# → dist/malaysia-terrain.pmtiles
# Duration: ~15 min
Dependencies
gdal 3.8+
rio-cogeo
python-gdal
pmtiles CLI
Source data
Copernicus DEM GLO-30
30m horizontal resolution
Free, global coverage
Output
PNG Terrain-RGB tiles
Zoom 5–14
~180 MB PMTiles
