2D Basemap Guide

Vector tile pipeline from OpenStreetMap PBF to PMTiles — plus Planetiler config, MapLibre style, and BM-first labels.

Build Steps

1

Fetch OSM Data

Downloads the latest Malaysia PBF from Geofabrik and verifies checksum.

scripts/fetch-osm.sh
# → data/malaysia-singapore-brunei-latest.osm.pbf
# Verifies md5 against Geofabrik manifest
2

Generate Vector Tiles

Planetiler converts PBF to PMTiles. Takes ~25 min on 8-core machine.

scripts/build-2d.sh
# → dist/malaysia-2d.pmtiles

# Or run Planetiler directly:
java -Xmx8g -jar planetiler.jar \
  --area=malaysia-singapore-brunei \
  --output=dist/malaysia-2d.pmtiles \
  --minzoom=0 --maxzoom=16 \
  --languages=ms,en --force
3

Validate Tile Count

Fails if tile count drops more than 5% vs. the previous build.

npm test -- --grep "tile-count"
# Checks tile count ±5% vs. previous build
# Checks all zoom levels 0–16 are populated
pmtiles verify dist/malaysia-2d.pmtiles
4

Upload to S3

Sets long cache TTL; CloudFront serves tiles globally.

scripts/upload.sh --layer 2d
# → s3://arah-tiles-prod/2d/malaysia-2d.pmtiles
# Sets: Cache-Control: public, max-age=86400

Tile Layers & Zoom Ranges

land
zoom 0–16  ·  landuse=*, natural earth
water
zoom 4–16  ·  natural=water, waterway=*
roads
zoom 6–16  ·  highway=*
buildings
zoom 13–16  ·  building=*
poi
zoom 12–16  ·  amenity=*, shop=*, tourism=*
labels
zoom 8–16  ·  name:ms → name:en → name

MapLibre Style

Colour Palette

Primary road — #E8B84B (Malay gold)
Water — #1A3A5C
Background light — #F2EFE9
Background dark — #0F1923
Accent / highway — #00D8A0

BM-First Labels

All label layers use coalesce: BM first, English fallback.

"text-field": [
  "coalesce",
  ["get", "name:ms"],
  ["get", "name:en"],
  ["get", "name"]
]

Editing the Style

# Edit
nano styles/arah-2d.json

# Validate
npm run validate-style

# Preview locally
docker run -v $(pwd)/dist:/data -v $(pwd)/styles:/styles \
  -p 8080:8080 maptiler/tileserver-gl-light
# → http://localhost:8080
