API Reference

REST + WebSocket API documentation for arah-api

Base URL & Authentication

Production
https://api.arah.my/v1
Development
http://localhost:3000
Authentication
Authorization: Bearer <firebase-id-token>
Success Response Envelope
{ "data": ..., "meta"?: ... }
Error Shape
{ "error": { "code": "...", "message": "...", "details"?: ... } }

Endpoints

/auth Authentication

Method Endpoint Auth Description
GET /auth/me ✓ Get current user profile
POST /auth/refresh ✓ Refresh session

/reports Reports

Method Endpoint Auth Description
POST /reports ✓ Submit a new community report
GET /reports — List active reports in bounding box
GET /reports/:id — Get single report
PUT /reports/:id/vote ✓ Upvote or downvote report
DELETE /reports/:id ✓ Remove own report

/route /geocode Routing & Geocoding (proxy)

Method Endpoint Auth Description
GET /route — Get route between two points
GET /geocode — Forward geocode address
GET /reverse-geocode — Reverse geocode coordinates

/profile User Profile

Method Endpoint Auth Description
GET /profile ✓ Get user profile + stats
PUT /profile ✓ Update profile
GET /profile/reports ✓ My submitted reports
DELETE /profile ✓ Delete account (PDPA)

/flood-zones /toll-costs … Flood & Malaysia APIs

Method Endpoint Auth Description
GET /flood-zones — Active flood zones
GET /toll-costs — Toll costs for route
GET /prayer-times — Waktu solat for coordinates
GET /checkpoints — JB-SG wait times

/health /metrics Health & Observability

Method Endpoint Auth Description
GET /health — Service health check
GET /metrics — Prometheus metrics

WebSocket Events

Client → Server

report:submit Submit a new report via WS
report:vote Vote on a report via WS
join:area Subscribe to a geohash area room
leave:area Unsubscribe from area room

Server → Client

report:new Broadcast when new report lands nearby
report:updated Vote count or status changed
report:expired Report removed from active index
flood:alert Emergency flood zone alert push

Rate Limits

60
req/min per IP
Public (unauthenticated) endpoints
120
req/min per user
Authenticated requests (Firebase UID)

Example Request 

curl https://api.arah.my/v1/reports?bbox=101.5,3.0,102.0,3.5 \
  -H "Authorization: Bearer YOUR_FIREBASE_ID_TOKEN"

Returns a paginated list of active reports within the bounding box ?bbox=minLng,minLat,maxLng,maxLat.
