Quick Start

git clone https://github.com/deanz93/arah-api.git
cd arah-api && npm ci
npm run dev  # http://localhost:3000

Base URLs

PROD https://api.arah.my/v1
DEV http://localhost:3000

All endpoints use Firebase ID token in Authorization: Bearer header.
