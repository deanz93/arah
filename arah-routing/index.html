Quick Start

Build the Docker image locally and test against the live Valhalla API.

git clone https://github.com/deanz93/arah-routing.git
cd arah-routing
docker build -t arah-routing:local .
docker run -p 8002:8002 -v $(pwd)/data:/data arah-routing:local
curl http://localhost:8002/status
First-run note: graph build takes ~15 minutes

The data/ directory must contain malaysia-latest.osm.pbf. Download from Geofabrik. Pre-built tiles are available on the EFS mount in staging/prod.
