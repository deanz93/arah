Kenapa Arah?

Waze dan Google Maps bukan untuk kita.

Lebih 90% pemandu Malaysia bergantung pada platform asing yang menjual data anda, mengabaikan keperluan tempatan, dan tidak bertanggungjawab kepada mana-mana pihak berkuasa Malaysia.

Tiada Kedaulatan Data

Lokasi GPS, sejarah laluan dan corak pemanduan anda mengalir ke pelayan di AS — tertakluk kepada undang-undang asing dan broker data. Arah menyimpan semua data di Malaysia.

UI Bukan Bahasa Malaysia

Panduan suara BM yang tidak tepat, UI dahulukan Bahasa Inggeris, tiada kesedaran tentang format alamat Malaysia. Arah dibina dengan Bahasa Malaysia dahulu, dari hari pertama.

Tiada Ciri Tempatan

Amaran banjir kilat? Kos tol dalam Ringgit? Amaran Zon Selamat? Pencari R&R PLUS? Tiada aplikasi asing yang menangani realiti pemanduan unik Malaysia. Arah lakukan semua itu.

Ciri Utama

Semua yang anda perlukan untuk memandu di Malaysia.

Dari panduan suara pusingan demi pusingan dalam Bahasa Malaysia hingga amaran banjir masa nyata — Arah direka untuk cara pemandu Malaysia sebenarnya.

Navigasi Pusingan demi Pusingan

Panduan suara pusingan demi pusingan dalam Bahasa Malaysia dan Bahasa Inggeris. 3 laluan alternatif — terpantas, bebas tol, dan terpendek. Laluan semula automatik apabila anda tersesat.

Suara BM Laluan Semula Auto Panduan Lorong

Laporan Komuniti Masa Nyata

Laporkan perangkap kelajuan polis, kemalangan, jalan berlubang, sekatan jalan raya dan lagi — dengan satu ketukan. Laporan disiar serta-merta kepada semua pemandu berhampiran melalui WebSocket.

WebSocket Auto-Luput 10 Jenis Laporan

Amaran Banjir Kilat

Eksklusif Malaysia. Kesan apabila laluan anda melalui kawasan banjir dan cadangkan laluan pemindahan secara automatik. Amaran FCM kecemasan kepada semua pengguna yang terjejas.

Geofencing FCM Push MVP

Kos Tol dalam Ringgit

Lihat kos tol tepat dalam RM untuk setiap laluan alternatif sebelum anda memandu. Pangkalan data plaza tol untuk semua lebuh raya PLUS, Lebuhraya, dan tol negeri.

DB Tol Per Plaza MVP

Peta Luar Talian

Muat turun kawasan peta penuh untuk kegunaan luar talian — Semenanjung Malaysia, Sabah, atau Sarawak. Navigasi tanpa internet. Kemas kini delta hanya memuat turun jubin yang berubah.

PMTiles Kemas Kini Delta Malaysia Timur

Privasi & Pematuhan PDPA

Tiada penjualan data. Ping GPS tanpa nama dengan cincangan HMAC sehala. Pematuhan PDPA 2010 sepenuhnya. Mod navigasi tanpa nama — tiada akaun diperlukan.

PDPA 2010 Tiada Jualan Data HMAC Anon

Untuk Malaysia, Dari Malaysia

Ciri-ciri yang tiada dalam Waze atau Google Maps —
kerana mereka tidak faham Malaysia.
01

Amaran Banjir + Laluan Pemindahan

Amaran kawasan banjir masa nyata dengan laluan semula automatik ke jalan pemindahan. Laporan banjir komuniti mencetuskan pemberitahuan tolak kecemasan kepada semua pemandu di kawasan terjejas.

MVP
02

Zon Selamat — Amaran Kawasan Sekolah

Amaran audio dan visual apabila mendekati kawasan sekolah semasa waktu persekolahan (7–8pg, 1–2ptg). Papar had laju 30km/j. Aktif hanya semasa penggal persekolahan.

MVP
03

Waktu Solat & Pencari Surau/Masjid

Papar waktu solat untuk lokasi semasa anda. Cari masjid atau surau terdekat sepanjang laluan anda. Pemberitahuan peringatan pilihan sebelum setiap waktu solat.

v1
04

Anggaran Trafik Balik Kampung

Ramalan trafik sejarah untuk musim perayaan — Hari Raya, Tahun Baru Cina, Deepavali. Amaran ETA yang lebih tinggi pada koridor KL–JB, KL–Penang.

v1
05

Pos Kawalan Johor–Singapura

Masa menunggu yang dilaporkan komuniti di pos pemeriksaan Woodlands dan Tuas. Corak masa melintas bersejarah mengikut hari dan masa. Cadangan "masa terbaik untuk melintas".

v1
06

R&R PLUS & Harga Petrol

Cari semua hentian R&R PLUS sepanjang laluan antara bandar. Harga RON95, RON97, dan diesel semasa di stesen petrol berdekatan (dikemas kini komuniti).

v1
07

Format Alamat Malaysia & Kod Pos

Carian menggunakan kod pos Malaysia (01000–98859). Format paparan yang betul: No. 12, Jalan Ampang, Taman Ampang Indah, 68000 Ampang, Selangor. Sabah dan Sarawak disokong sepenuhnya.

MVP
08

Suara Panduan Bahasa Malaysia

Panduan suara TTS penuh dalam Bahasa Malaysia semula jadi — "Dalam 200 meter, belok kiri di Jalan Ampang." Bukan terjemahan — direka dan diuji untuk nama jalan Malaysia.

Tindanan Teknologi

Dibina atas piawaian terbuka. Tiada kebergantungan proprietari.

Setiap komponen adalah sumber terbuka. Tiada Google Maps API. Tiada Mapbox. Tiada data asing. Berjalan sepenuhnya pada infrastruktur AWS rantau Malaysia.

Mudah Alih
React Native 0.74
iOS + Android dari satu asas kod TypeScript. MapLibre GL untuk rendering. Zustand + TanStack Query.
Portal Web
Next.js 14
Papan pemuka admin dengan App Router + Server Components. Firebase Admin SDK sisi pelayan. Tailwind CSS.
Pintu Masuk API
Fastify + Node.js
REST API bertaip + WebSocket Socket.io. Firebase Auth. Caching Redis. Pengesahan Zod. Had kadar 60 req/min.
Enjin Laluan
Valhalla
Enjin laluan C++ yang dihoskan sendiri. Graf OSM Malaysia ~4GB RAM. Manoeuvre pusingan demi pusingan, pengelakan tol, 3 alternatif.
Pengekod Geo
Nominatim 4.4
Carian alamat OSM dihoskan sendiri. countrycodes=my dikuatkuasakan. Format alamat & kod pos Malaysia. Import 30 minit.
Jubin Peta
PMTiles + TileServer-GL
Jubin vektor pada CDN CloudFront. Had Malaysia 99.5,1.0–119.5,7.5. Cache CDN 7 hari. Tiada kos API per permintaan.
Pelayan
Firebase
Auth (Google + OTP Telefon), Firestore DB masa nyata, Analitik, Crashlytics, Cloud Functions v2.
Infrastruktur
Kubernetes / AWS EKS
Terraform IaC. ap-southeast-1 (Singapura). ElastiCache Redis 7.1. Auto-skala HPA. CI/CD GitHub Actions.
