Built for Malaysians

Everything a Malaysian driver actually needs — not an afterthought.

Turn-by-Turn Navigation

Valhalla-powered routing with Bahasa Malaysia voice TTS. 3 route options — fastest, toll-free, and shortest — with real-time rerouting.

📡

Community Reports

Real-time incident reports via WebSocket. 10 report types including banjir kilat, kemalangan, and roadblock — 1-tap submit with photo support.

🇲🇾

Malaysia-First Features

Flash flood evacuation routes, Zon Selamat school zones, waktu solat overlay, toll cost display in RM, and Jawi label toggle.

Quick Start

Get the app running locally in 3 steps.

# 1. Clone the repo
git clone https://github.com/deanz93/arah-mobile.git
cd arah-mobile

# 2. Install dependencies
npm ci

# 3. Run on Android (or swap for run-ios)
npx react-native run-android

Need more detail? See Contributing for full environment setup, iOS steps, and Android emulator configuration.
