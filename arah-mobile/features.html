NativeWind-First Styling

All UI components in arah-mobile are built with NativeWind v4 (Tailwind CSS for React Native). No StyleSheet.create() allowed — contributors must use Tailwind utility classes exclusively. This keeps styling consistent, reviewable, and easy to theme.
