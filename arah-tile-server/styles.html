Map Styles

Vector tile styles for the Arah navigation app

Style Overview

Two map styles are provided: arah-day and arah-night. Both are MapLibre GL compatible and follow the Mapbox GL Style Specification. Bahasa Malaysia labels are enabled by default; English and Jawi labels can be toggled at runtime via the layer visibility API.

Style Endpoints

GET /styles/arah-day/style.json

Returns the full MapLibre GL style JSON for the daytime theme.

GET /styles/arah-night/style.json

Returns the full MapLibre GL style JSON for the night theme.

Tile Specification

Tiles follow the MapLibre/Mapbox Vector Tile Specification 2.1. Compatible with MapLibre GL JS and MapLibre GL React Native. Use with the style endpoints above to render a fully styled map.
