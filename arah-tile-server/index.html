Planetiler Build

OSM PBF → PMTiles pipeline with Malaysia bounding box coverage, reproducible Docker builds, and compact output.

  • ✓ Malaysia bbox: 99.5,1.0–119.5,7.5
  • ✓ Reproducible Docker-based builds
  • ✓ ~500 MB PMTiles output
  • ✓ Automated monthly refresh

TileServer-GL

High-performance vector tile serving with day and night styles, bilingual labels, and Jawi script support.

  • ✓ Day + night map styles
  • ✓ Bilingual BM/EN labels
  • ✓ Jawi script layer (RTL)
  • ✓ p95 < 200ms cached

CDN Delivery

CloudFront-fronted tile delivery with atomic EFS updates, long-lived caching, and secure non-root container.

  • ✓ CloudFront distribution
  • ✓ 7-day Cache-Control headers
  • ✓ Atomic EFS tile update
  • ✓ Non-root Docker uid 1001

Quick Start

git clone https://github.com/deanz93/arah-tile-server.git
cd arah-tile-server
docker build -t arah-tile-server:local .
docker run -p 8080:8080 -v $(pwd)/data:/data arah-tile-server:local
curl http://localhost:8080/health

Tile URL Pattern

Production tile endpoint served via CloudFront CDN:

https://tiles.arah.my/data/malaysia/{z}/{x}/{y}.pbf

Data Pipeline

↓

4 CloudFront invalidation
