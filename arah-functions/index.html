Quick Start

git clone https://github.com/deanz93/arah-functions.git
cd arah-functions && npm ci
firebase emulators:start  # local dev

Emulator Suite

Functions localhost:5001
Firestore localhost:8080
Auth localhost:9099
Pub/Sub localhost:8085

All tests run against the emulator suite. No production Firebase needed for dev.
