Quick Start

Clone the repo and initialise the VPC Terraform module to get started.

git clone https://github.com/deanz93/arah-infra.git
cd arah-infra
terraform -chdir=terraform/modules/vpc init
terraform -chdir=terraform/modules/vpc plan -var-file=../../vars/staging.tfvars
AWS Region: ap-southeast-1 (Singapore)

All resources are provisioned in AWS ap-southeast-1 to minimise latency for Malaysian users. Ensure your AWS credentials are configured for this region before running Terraform.
