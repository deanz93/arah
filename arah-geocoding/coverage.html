API Examples

Forward geocode (address → coordinates)

curl 'http://localhost:7070/search?q=KLCC+Kuala+Lumpur&countrycodes=my&format=jsonv2'

Postcode search

curl 'http://localhost:7070/search?q=50450&countrycodes=my&format=jsonv2'

Reverse geocode (coordinates → address)

curl 'http://localhost:7070/reverse?lat=3.1570&lon=101.7123&format=jsonv2'

East Malaysia — Kota Kinabalu

curl 'http://localhost:7070/search?q=Kota+Kinabalu+Sabah&countrycodes=my&format=jsonv2'

Response Format

Example Malaysian address in jsonv2 format:

{
  "display_name": "KLCC, Jalan Ampang, Kuala Lumpur City Centre, 50088 Kuala Lumpur, Wilayah Persekutuan Kuala Lumpur",
  "lat": "3.15784",
  "lon": "101.71186",
  "address": {
    "attraction": "KLCC",
    "road": "Jalan Ampang",
    "suburb": "Kuala Lumpur City Centre",
    "postcode": "50088",
    "city": "Kuala Lumpur",
    "state": "Wilayah Persekutuan Kuala Lumpur",
    "country": "Malaysia",
    "country_code": "my"
  }
}
