Malaysian Address Search

Full-text address lookup for all 16 Malaysian states with BM name search and East Malaysia coverage.

  • ✓ All 16 states covered
  • ✓ Postcode 01000–98859
  • ✓ BM name search
  • ✓ East Malaysia (Sabah, Sarawak)

Reverse Geocoding

Coordinates → formatted Malaysian address, with zoom-level awareness and Redis caching for fast repeat lookups.

  • ✓ Coordinates → address
  • ✓ Zoom-level aware
  • ✓ 1h Redis cache
  • ✓ p95 < 150ms

PDPA Compliant

No PII stored in logs. Query hash logging only, with enforced Malaysian countrycodes and graceful 5-second timeout.

  • ✓ No PII in logs
  • ✓ 5-second timeout + BM error
  • ✓ countrycodes=my enforced
  • ✓ Query hash logging only

Quick Start

git clone https://github.com/deanz93/arah-geocoding.git
cd arah-geocoding
docker compose up nominatim  # starts pre-built image
curl 'http://localhost:7070/search?q=KLCC&countrycodes=my&format=jsonv2'

Live Endpoint

Production geocoding endpoint:

https://geo.arah.my/search?q=...&countrycodes=my&format=jsonv2
